chef-swap
=========

## configure swap for you centos

## Usage

<pre>
{
    "name": "swap",
    "description": "swap configure",
    "default_attributes": {
    },
    "json_class": "Chef::Role",
    "run_list": [
        "recipe[swap]"
    ],
    "chef_type": "role",
    "override_attributes": {
    }
}
</pre>
